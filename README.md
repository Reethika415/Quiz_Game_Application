# Quiz Game Application 🎯

A command-line based C++ Quiz Game focused on OOP (Object Oriented Programming) concepts.

## Features
- 10 Multiple Choice Questions
- Each question carries 10 marks
- Total score: 100 marks
- Input validation (no crashes on wrong input)
- Personalized experience (asks your name & age)
- Pass/Fail result at the end

## Topics Covered
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Virtual Functions
- Constructor Overloading
- Access Specifiers

## How to Compile & Run
```bash
g++ quiz_game.cpp -o quiz_game
./quiz_game
